# AXIS_BANK_ANALYTICS_DASHBOARD_WITH_AWS_INTEGRATION
Dashboard for customers and managers in Axis bank using Streamlit and AWS cloud 

##  Overview
Banking analytics system built using:
- FastAPI (Backend)
- PostgreSQL (Database)
- Streamlit (Frontend)
- AWS S3 (PDF ingestion)

##  Features
- PDF bank statement ingestion
- Regex-based transaction extraction
- Duplicate protection
- Negative balance handling
- Branch-level KPI comparison

##  Tech Stack
Python, FastAPI, Streamlit, PostgreSQL, AWS S3, Plotly

##  Future Enhancements
- LLM integration using LangChain
- AI-powered financial insights
- Deployment on AWS
